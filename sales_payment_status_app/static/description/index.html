Odoo Sale Payment Status App

Odoo Payment Status on Sales App offers users a convenient platform to monitor invoice and payment statuses within sale orders. This Odoo app enables users to access detailed information on invoice status and track payment progress. Users can conveniently register payments directly within the sale order. This app also includes an intuitive invoice status filter in the tree view, enhancing user experience by facilitating efficient data management. Additionally, users can gain valuable insights into payment details directly within the sale order interface, ensuring a comprehensive overview of financial transactions.

Invoice Payment Status In Sale Order

We provide different payment status in sale order like no invoice status for sale order has no invoice, not paid status for all invoice has not paid, partial paid status for any invoice has partially paid, overdue status for any invoice has overdue, fully paid status for all invoice has paid.
